Compartilhando meu pipeline de treinamento que levou 6 meses para desenvolver:

• Simula trades com trailing stop
• Calcula win/loss para cada candle
• Balanceamento de classes

LightGBM (Local):
• Rápido: ~10 min para 7 modelos

• Intensivo: ~3 horas

• Combina LightGBM (85%) + LSTM (15%)
• Weighted voting
• Score mínimo: 28%
• Resultado: Mais robusto que modelos isolados

📊 ETAPA 6: Backtesting
• 36.000+ candles
• Simula spread, slippage
• Métricas: Precision, Recall, F1
